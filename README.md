#  Sistema de Cadastro Eleitoral em Python

Projeto desenvolvido com foco em prática de lógica de programação, manipulação de arquivos e organização de dados estruturados em Python.

---

##  Descrição

Este sistema simula um cadastro de usuários com geração de dados do título de eleitor.  
Os dados são armazenados em arquivo `.txt`, organizados em blocos delimitados, permitindo busca estruturada por CPF.

O projeto foi desenvolvido com foco em aprendizado prático e consolidação de conceitos fundamentais da programação.

---

##  Objetivo do Projeto

- Praticar manipulação de arquivos em Python
- Trabalhar com estruturas condicionais e laços de repetição
- Implementar busca por bloco em arquivos texto
- Aplicar tratamento de exceções
- Desenvolver pensamento lógico estruturado

---

##  Funcionalidades

- Cadastro de usuário  
- Geração de dados do título de eleitor  
- Armazenamento em arquivo texto  
- Busca de cadastro por CPF  
- Tratamento de erro para arquivo inexistente  

---

##  Conceitos Aplicados

- Estruturas condicionais (`if`, `else`)
- Laços de repetição (`for`)
- Manipulação de listas
- Processamento de dados por delimitador
- Tratamento de exceções (`try/except`)
- Organização lógica de dados em blocos

---

##  Tecnologias Utilizadas

- Python 3
- Manipulação de arquivos `.txt`
- Git & GitHub para versionamento
